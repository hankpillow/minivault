🔒 Minivault

Set a password with your friends/team and use it to encrypt data and share securely.

It uses the built-in crypto API available on the browser - zero external services or internet required (using the browser extension).

No data/cookie is kept or sent by the vault!
👉 Try here 🌐 Chrome Extension 🦊 Firefox Add-Ons 🐋 Docker
docker run minivault/vault encrypt "foo" "bar" docker run minivault/vault decrypt "foo" "bar"
