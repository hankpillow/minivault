🔒 Minivault

The Minivault is a web app/browser extension designed to help to share sensitive data securely.

Set a secret with your friends/team and use it to encrypt and share your data!

No data/cookie is kept or sent by the vault!
It uses only the native Crypto API available in the browser - zero external services or internet required (using the browser extension).
👉 Try the Minivault 🌐 Chrome Extension 🦊 Firefox Add-Ons

For devs:

🐋 Docker
docker run minivault/vault encrypt "foo" "bar" docker run minivault/vault decrypt "foo" "bar"
